# Surplus-Electricity Auxiliary Tanks

Interactive static prototype for simulating surplus-electricity pre-heating with an auxiliary hot water tank.

Live preview: https://paulmvpbuild.github.io/surplus-electricity-auxiliary-tanks/

## Features

- NEM live data-style price chart
- Draggable price set point and time bar
- Auxiliary tank thermostat set point
- Heating status and energy-flow animation
- Shower drain/refill interaction between the regular and auxiliary tanks

## Local Preview

```sh
python3 -m http.server 5173
```

Then open http://localhost:5173/.
